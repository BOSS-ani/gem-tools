ツールの使用方法とヒント

【何をするツール？】

このツールは、競馬分析AI「NotebookLM」で質の高い分析結果を得るための、最適な指示書（プロンプト）を自動で生成します。

【基本的な使い方】

  1. 分析情報の入力: 「ステップ1」で分析したい「レース名」、分析の切り口となる「分析のタイプ」、そして特に掘り下げたい「重視する点」をフォームに入力・選択します。
  2. プロンプト生成: 「プロンプトを生成」ボタンを押すと、入力内容に基づいた最適な指示書が「ステップ2」のボックスに生成されます。
  3. コピー: 「コピー」ボタンで、生成されたプロンプトをクリップボードにコピーします。

【フォローアップ（ワークフロー）】

コピーしたプロンプトを、NotebookLMの入力欄に貼り付けて分析を開始してください。このツールで生成された質の高い指示書を使うことで、NotebookLMはあなたの意図を正確に理解し、より深く、的確な分析結果を出力します。分析結果（テキスト）が得られたら、次の「コンテンツ生成」ツールでそのテキストを活用し、ウェブページやナレーション原稿を作成しましょう。

ツールの使用方法とヒント

【何をするツール？】

このツールは、入力された「分析テーマ」と「分析データ」を基に、段階的なプロセスで関連コンテンツを生成します。まずWeb検索を活用してHTMLのウェブページを作成し、次にそのウェブページを元にナレーション原稿とスライドを生成します。

【基本的な使い方】

  1. データ入力: 「ステップ1」で分析テーマと基礎となるデータを入力します。
  2. Webページ生成: 「① Webページを生成」ボタンを押します。AIがWeb検索を行い、分析に基づいたHTMLを「ステップ2」に生成します。
  3. 追加コンテンツ生成: 「ステップ2」に表示されたHTMLを元に、「② ナレーションを生成」または「③ スライドを生成」ボタンを押してください。結果は「ステップ3」の各ボックスに表示されます。
  4. 結果の活用: 各ボックスの「コピー」ボタンを使って、生成されたコンテンツをそれぞれの用途に合わせて活用します。

【フォローアップ（ワークフロー）】

  • ウェブページ (HTML): ブログ記事やウェブサイトの1ページとしてそのまま利用できます。
  • ナレーション原稿: 生成された原稿は、このハブの「AIナレーション」ツールに貼り付けて、高品質な音声ファイルを作成するのに最適です。
  • スライド化コード: 生成されたHTMLコードをテキストエディタに貼り付けて`.html`ファイルとして保存し、ブラウザで開けばプレゼンテーションとして使用できます。

ダウンロード (WAV)

ツールの使用方法とヒント

【何をするツール？】

このツールは、入力された日本語のテキスト原稿から、AIによる高品質なナレーション音声（WAV形式）を生成します。

【基本的な使い方】

  1. 原稿入力: 「ステップ1」のボックスに、ナレーションさせたい文章を貼り付けます。
  2. 読み方生成: 「概要と読み方を生成」ボタンを押すと、AIが原稿の要約と、漢字の読み方をカタカナに変換したものを生成します。
  3. 読み方チェック＆修正: 「ステップ3」で、AIが生成したカタカナの読み方を確認します。特に人名や専門用語など、意図と違う読み方になっている場合は、このボックス内で直接カタカナを修正してください。ここでの修正が、最終的な音声の品質を決定する最も重要な工程です。
  4. 音声化＆ダウンロード: 読み方の修正が完了したら、「音声に変換」ボタンを押します。生成された音声を「ステップ4」でプレビュー再生し、問題がなければ「ダウンロード」ボタンでPCに保存します。

【フォローアップ】

ダウンロードした音声ファイルは、動画編集ソフトに取り込んで映像に合わせるなど、様々なコンテンツ制作にご活用いただけます。

ツールの使用方法とヒント

【何をするツール？】

このツールは、アップロードされた一つの音声ファイル（MP3）を、指定した数のパートに自動で分割し、それぞれを個別のMP3ファイルとしてダウンロードできるようにします。

【基本的な使い方】

  1. 分割数を設定: 「ステップ1」で、音声をいくつのパートに分けたいか数字で入力し、「セクションを生成」ボタンを押します。下にその数だけ操作パネルが表示されます。
  2. 音声ファイルをアップロード: 「ステップ2」で、分割したいMP3音声ファイルを選択します。
  3. 時間を確認・調整: ファイルを読み込むと、各パートの開始時間と終了時間が自動で計算されます。もし特定の時間で区切りたい場合は、秒数を手動で修正することも可能です。
  4. ダウンロード: 各パートのパネルにある「ダウンロード」ボタンを押すと、その部分だけが切り出されたMP3ファイルがPCに保存されます。

【フォローアップ（ワークフロー）】

例えば、「AIナレーション」ツールで作成した一つの長いナレーション音声を、動画のスライドやシーンの数に合わせてここで分割することができます。分割した音声ファイルは、動画編集ソフトで各シーンに配置する際に便利です。最終的に、次の「YouTube概要欄」ツールで動画公開の準備をしましょう。

ツールの使用方法とヒント

【何をするツール？】

このツールは、あなたのYouTube動画（特に競馬予想）に最適化された、魅力的でクリックしたくなる「タイトル」と「概要欄」をAIが自動で生成します。

【基本的な使い方】

  1. 動画情報の入力: 「ステップ1」のフォームに、動画のキーポイントとなる情報を入力します。AIに動画を分析させた結果を貼り付けると効率的です。
  2. 独自メッセージの追加: 「概要欄に必ず含めたい内容」の欄に、その動画ならではの特別なメッセージや視聴者への呼びかけなどを自由に追加できます。
  3. 生成: 「タイトルと概要欄を生成」ボタンを押すと、AIがWeb検索も行い、最新情報やレースの格付けなどを加味した上で、最適なコンテンツを「ステップ2」に生成します。
  4. コピー＆ペースト: 各ボックスの「コピー」ボタンを使い、生成されたタイトルと概要欄をYouTubeのアップロード画面に貼り付けます。

【フォローアップ（ワークフロー）】

このツールは、コンテンツ制作フローの最終段階です。ここで生成されたタイトルと概要欄を使えば、視聴者の興味を引き、クリック率やエンゲージメントの向上が期待できます。ハッシュタグなども自動で最適化されますので、そのままYouTubeでご活用ください。
